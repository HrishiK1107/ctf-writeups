# OverTheWire: Bandit

Bandit is a Linux-focused wargame that teaches fundamental skills required for cybersecurity professionals. The challenges gradually introduce concepts such as file handling, permissions, encoding, networking, service enumeration, encryption, SSH authentication, and privilege escalation.

This repository documents my solutions, thought process, and lessons learned while progressing through the Bandit levels.
